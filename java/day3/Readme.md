## Boolean 

It can only take these values true or false.

```

boolean isAdult = true;
boolean isQualified = false;
```

## character

the char data type is used to store single character. It is surrounded by single quote.

```

char myGrade = 'A';
```
## strings

string in java is actually is non-primitive datatype, because it refers to an object.

```

string firstName = "Manoranjan";
```

### Non-Primitivve datatypes
- These are created by the programmer using primitive data types(Except string).

- It can be used to call methods to perform certain operations.

- Typically non-primitive data types starts with uppercase letter.
- non-primitive data can be null.


## var Keyword
- The var keyword lets the compiler automatically detect the type of a variable based on the value you assign to it.
