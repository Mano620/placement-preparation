## OPERATORS
- Arithmetic operators
- Assignment operators
- Comparison operators
- Logical operators
- Bitwise operators

## Arithmatic operators
- Operator	       Name	          Description	                            Example
-   +	           Addition	      Adds together two values	                x + y
-   -	           Subtraction	  Subtracts one value from another	        x - y
-    *	           Multiplication Multiplies two values                     x * y
-    /	           Division	      Divides one value by another	            x / y
-    %              Modulus       Returns the division remainder	        x % y
-    ++	           Increment      Increases the value of a variable by 1	++x
-    --	           Decrement	  Decreases the value of a variable by 1	--x


### Assignment operators
- Operators are used to assign values to variable.
Example -
- Assigment (=)
- Addition Assigment (+=)
- Substraction Assigment (-=)
- Multiplication Assigment (*=)
- Division Assigment (/=)
- Modules Assigment (%=)
- AND Assigment (&=)
- OR Assigment (|=)
- XOR Assigment (^=)
- Left Shift Assigment (<<=)
- Right Shift Assigment (>>=)



## Comparison Operator

Comparison operator are used to compare two values(or variable).
It return value of a comparision is either true or false.
- Equal to (==)
- Not Equal to (!=)
- Greater than(>)
- less than (<)
- Greater than or equal to (>=)
- Less than or equal to(<=)

## Logical operators
- As with comparison operators,  we can also test for true or false values with logical operators.

- AND(&&)
- OR (||)
- NOT(!)

## Bitwise Operator
- It helps in performing operation directly on binary bits(0s and 1s ) of integers types.

- AND (&)
- OR (|)
- XOR (^)
- NOT (~)
- LEFT SHIFT (<<)
- RIGHT SHIFT(>>)
